== Description ==

This plugin lists last updated threads of a vBulletin's DB into Wordpress.

* It's works with 2.x and 3.x versions of vBulletin.
* It's fully configurable via Control Panel.

== Installation ==

1. Upload `vb_reader.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

See also: http://www.minimal-software.com/?page_id=68

== Frequently Asked Questions ==

Can you the change where the threads are listed in WordPress automatically ?

 No you can't. The threads will be listed on your WordPress' home page only. You'll need to edit the plugin if you want to change that.

With wich versions of vBulletin this plugin works ?

 With 2.x (tested on 2.2.6) and with 3.x (tested on 3.6.8).

Can I see a website with the plugin running ?

 Yes you can: http://www.plgrosario.com.ar
